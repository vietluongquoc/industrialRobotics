# Industrial Robotics
This is the python application to control a robot arm in course "Industrial Robotics"

## Requirement
- python 3.12
- i2cpy
- Qt5
- pca 9685
- usbi2c 341T

## How to run
- install requirement
```
python -m venv venv
pip install i2cpy
pip install Qt5
python main.py

```
